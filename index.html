Multiplayer Gobang Based on Reinforcement Learning

We proposed a multiplayer (more than two players) Gobang agent based on modified Alpha MCTS consists of transformer blocks implemented with Attention mechanism. We also compared the performance of each architecture.

Image Inpainting Using GAN, Partial Convolution and Region Normalization

We proposed a network that combines CGAN, Partial Convolution, and Region Normalization dealing with image inpainting tasks. Our model outperforms the baseline model both qualitatively and quantitatively.

Reimplementation of Swin Transformer

We mainly investigate the trending computer vision network Swin transformer and its variants.We evaluated our models with an image classification task as well as a downstream classification task on a private LCD screen defect dataset.
